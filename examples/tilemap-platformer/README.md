# TileMap Platformer

Рабочий пример TileMap v2:

- inline tileset canvas
- `TileMap.fromJSON`
- `TileMapComponent`
- `TileMapColliderComponent.resolveAABB`
- `CameraComponent` follow

Управление: A/D или стрелки, Space/W/Up для прыжка.

Запуск: `npm run develop`, затем открыть `/examples/tilemap-platformer/`.
