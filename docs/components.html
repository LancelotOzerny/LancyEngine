Раздел 5: Компоненты и их применение

Разбор всех компонентных классов: что делают, где полезны, как подключать в GameObject.

Главная Назад Дальше

TransformComponent

Хранит позицию и поворот объекта.

obj.transform.setPosition(120, 80);
obj.transform.translate(16, 0);
obj.transform.rotation = 25;

SpriteComponent

Рисует спрайт из ассетов или цветной прямоугольник.

obj.bindComponent(new SpriteComponent({
  sprite: "player", // ключ файла player.png
  width: 96,
  height: 96
}));

// вариант без изображения
obj.bindComponent(new SpriteComponent({
  width: 96,
  height: 96,
  color: "#38bdf8"
}));

AnimationComponent

Sprite sheet анимация + state machine (idle/run/jump и переходы).

const anim = new AnimationComponent({
  sprite: "hero_sheet",
  frameWidth: 64,
  frameHeight: 64,
  initialState: "idle",
  states: {
    idle: {
      fps: 6,
      frames: [0, 1, 2, 3],
      transitions: [{ to: "run", param: "isMoving", equals: true }]
    },
    run: {
      fps: 12,
      frames: [8, 9, 10, 11, 12, 13],
      transitions: [{ to: "idle", param: "isMoving", equals: false }]
    }
  }
});

obj.bindComponent(anim);
anim.setParam("isMoving", true);
Инструкция: 1) загрузите sheet в Engine.run({ images: [...] }), 2) задайте frameWidth/frameHeight, 3) опишите states и transitions, 4) обновляйте параметры через setParam() в update(dt).

ColliderComponent

AABB-коллизии, слои и маски столкновений, контактные флаги.

const col = new ColliderComponent({
  width: 48,
  height: 48,
  layer: 1,
  mask: [2, 3],
  isTrigger: false
});
obj.bindComponent(col);

if (col.isCollision) {
  console.log("Столкновение с:", col.collision?.parent?.name);
}

PhysicComponent

Скорость, силы, гравитация, drag, осевой резолв столкновений.

const body = new PhysicComponent({
  gravityScale: 2.0,
  drag: 0.12
});
obj.bindComponent(body);

body.applyForce(0, -1800);
body.velocity.y = -520;
Для реакции на землю используйте body.isGrounded.

CameraComponent

Следящая камера с offset, dead-zone, ограничениями и shake.

const camHost = new GameObject();
const camera = new CameraComponent({
  isMain: true,
  target: player,
  smoothness: 10,
  deadZoneWidth: 320,
  deadZoneHeight: 180,
  bounds: { left: 0, top: 0, right: 5000, bottom: 2000 }
});
camHost.bindComponent(camera);
scene.gameObjects.append(camHost);

camera.shake(0.25, 12);

Сводка по компонентам

Компонент Когда использовать
TransformВсегда, базовая трансформация объекта.
SpriteНужно визуально отрисовать объект.
AnimationНужна покадровая анимация и FSM-переходы состояний.
ColliderНужны столкновения, триггеры, контакты.
PhysicНужно движение по скорости/силам/гравитации.
CameraНужна динамическая камера за игроком/объектом.
