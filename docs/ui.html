Раздел 6: UI поверх Canvas

UI строится на DOM и работает независимо от world-координат canvas.

Главная Назад Дальше

Классы UI

  • UiPanel — корневой контейнер интерфейса.
  • UiText — текстовый элемент.
  • UiButton — кнопка с callback.
  • UiImage — изображение.
  • UiElement / UiTextElement — базовые слои абстракции.

Пример HUD

const hud = new UiPanel("HUD", {
  width: "100%",
  height: "100%"
});
hud.backgroundColor = "transparent";
hud.index = 10;

const scoreText = hud.createText({
  text: "Score: 0",
  anchors: "left top",
  x: 20,
  y: 16
});

hud.createButton({
  text: "Pause",
  anchors: "right top",
  x: 20,
  y: 16,
  click: () => Engine.instance.stopGameLoop()
});

Якоря и позиционирование

anchors Как работает
left topОтступы от левого верхнего угла.
right topОтступы от правого верхнего угла.
center centerЦентрирование по обеим осям.
left bottomОтступы от левого нижнего угла.
UI — это DOM-слой, поэтому он не масштабируется world-камерой, что удобно для HUD и меню.
