Минимальный запуск игры

const engine = Engine.instance;

engine.init("#game", {
  width: 1280,
  height: 720,
  backgroundColor: "#0f1720",
  screenMode: "contain"
});

class MainScene extends Scene {
  init() {
    const block = new GameObject();
    block.bindComponent(new SpriteComponent({
      width: 120,
      height: 120,
      color: "#22c55e"
    }));
    block.transform.setPosition(220, 220);
    this.gameObjects.append(block);
  }
}

engine.scenes.append(new MainScene());
engine.run(() => {}, { images: [], audio: [] });

Что проверить после запуска

  1. Canvas отображается и занимает ожидаемую область.
  2. Объект из сцены виден на экране.
  3. Нет ошибок в консоли по загрузке скриптов.
  4. Если подключены ассеты, ключи совпадают с именами файлов без расширений.
