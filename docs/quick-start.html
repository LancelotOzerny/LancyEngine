Минимальный запуск игры

const engine = Engine.instance;

engine.init("#game", {
  width: 1280,
  height: 720,
  backgroundColor: "#0f1720",
  screenMode: "contain"
});

class MainScene extends Scene {
  init() {
    const block = new GameObject();
    block.bindComponent(new SpriteComponent({
      width: 120,
      height: 120,
      color: "#22c55e"
    }));
    block.transform.setPosition(220, 220);
    this.gameObjects.append(block);
  }
}

engine.scenes.append(new MainScene());
engine.run(() => {}, { images: [], audio: [] });

Быстрый старт анимации (sprite sheet)

  1. Добавьте sprite sheet в images при Engine.run.
  2. Подключите AnimationComponent к объекту и задайте frameWidth/frameHeight.
  3. Опишите состояния в states (например idle и run).
  4. В update(dt) обновляйте параметры через anim.setParam(...).
const player = new GameObject();
player.bindComponent(new AnimationComponent({
  sprite: "hero_sheet",
  frameWidth: 64,
  frameHeight: 64,
  initialState: "idle",
  states: {
    idle: {
      fps: 6,
      frames: [0, 1, 2, 3],
      transitions: [{ to: "run", param: "isMoving", equals: true }]
    },
    run: {
      fps: 12,
      frames: [8, 9, 10, 11, 12, 13],
      transitions: [{ to: "idle", param: "isMoving", equals: false }]
    }
  }
}));

Что проверить после запуска

  1. Canvas отображается и занимает ожидаемую область.
  2. Объект из сцены виден на экране.
  3. Нет ошибок в консоли по загрузке скриптов.
  4. Если подключены ассеты, ключи совпадают с именами файлов без расширений.
  5. Для анимации: проверьте, что кадры переключаются и переходы состояний срабатывают.
