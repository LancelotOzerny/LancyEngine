Раздел 1: Описание игрового движка

LancyEngine — легкий 2D-движок на JavaScript + Canvas с ECS-подходом, физикой, коллизиями, камерой, загрузкой ассетов, аудио-менеджером и DOM UI поверх canvas.

Что умеет движок

Game loop 60 FPS Scene / GameObject / Component AABB-коллизии Arcade-физика Камера с dead-zone Sprite sheet animations AudioManager DOM UI

Типичные сценарии

  • Аркады и платформеры.
  • Прототипы gameplay-механик.
  • Учебные проекты по архитектуре игровых систем.

Плюсы и недостатки

Плюсы

  • Простая структура исходников и низкий порог входа.
  • Четкая декомпозиция на core/компоненты/UI.
  • Готовые базовые системы: коллизии, физика, камера, аудио.
  • Удобен для быстрых итераций и обучения.

Ограничения

  • Коллизии только AABB (без сложной геометрии).
  • Физика arcade-уровня, без high-end симуляции.
  • SceneManager работает в модели одной активной сцены (без built-in scene stack).
  • EventSystem минималистичный: без namespaced-событий и middleware.
Движок лучше всего подходит для 2D проектов и курсовых задач, где важна прозрачность кода, а не максимальный объем готовых инструментов.
