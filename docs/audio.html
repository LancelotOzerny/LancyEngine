Раздел 7: AudioManager

Полный разбор звуковой системы: каналы, музыка, sfx, mute, pause и управление громкостью.

Главная Назад Дальше

Назначение AudioManager

AudioManager доступен как Engine.instance.audio. Управляет двумя типами воспроизведения: music и sfx, плюс общий канал master.

Канал Для чего
masterОбщий контроль громкости/паузы/мута.
musicФоновая музыка (один текущий трек).
sfxЗвуковые эффекты (могут звучать параллельно).

Базовый пример использования

await Engine.instance.run(() => {
  Engine.instance.audio.playMusic("theme", {
    loop: true,
    volume: 0.45
  });

  window.addEventListener("keydown", (event) => {
    if (event.code === "Space") {
      Engine.instance.audio.playSfx("jump", { volume: 0.8 });
    }
  });
}, {
  images: [],
  audio: ["./assets/theme.mp3", "./assets/jump.wav"]
});

Ключевые API-методы

  • playMusic(key, options) / stopMusic()
  • playSfx(key, options) / stopSfx()
  • setVolume(channel, value)
  • mute(channel) / unmute(channel) / toggleMute(channel)
  • pause(channel) / resume(channel)
  • stopAll()
Engine.instance.audio.setVolume("master", 0.8);
Engine.instance.audio.setVolume("music", 0.5);
Engine.instance.audio.toggleMute("sfx");
Engine.instance.audio.pause("master");
Engine.instance.audio.resume("master");

Практические замечания

  • Ключи берутся из имен файлов без расширений.
  • Музыку лучше запускать после пользовательского действия (клик/клавиша).
  • Для разных сцен удобно регулировать каналы, а не каждый клип вручную.
