Обновление: AudioManager в документации

Раздел аудио вынесен в отдельную страницу и дополнительно отражен в core-доках. Ниже минимальный пример запуска музыки и эффектов. 

await Engine.instance.run(() => {
  Engine.instance.audio.playMusic("theme", {
    loop: true,
    volume: 0.5
  });

  window.addEventListener("keydown", (event) => {
    if (event.code === "Space") {
      Engine.instance.audio.playSfx("jump", { volume: 0.8 });
    }
  });
}, {
  images: [],
  audio: ["./assets/theme.mp3", "./assets/jump.wav"]
});
Ключ аудио-ассета берется из имени файла без расширения: theme.mp3 → theme.
